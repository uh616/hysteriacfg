## 📥 Как использовать

### Hysteria2 клиент

1. Откройте настройки клиента
2. Добавьте подписку (выберите нужную ссылку выше)
3. Конфиги будут автоматически обновляться

### V2RayN / Nekoray / Clash

1. Скопируйте ссылку на подписку
2. Вставьте в настройках подписок
3. Обновите подписку
