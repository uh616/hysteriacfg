# Быстрый старт

## Windows

1. Запустите `setup.bat` для установки всех зависимостей
2. Запустите `run.bat` для запуска проверки

## Linux/Mac

1. Сделайте скрипты исполняемыми:
   ```bash
   chmod +x setup.sh run.sh
   ```

2. Запустите установку:
   ```bash
   ./setup.sh
   ```

3. Запустите проверку:
   ```bash
   ./run.sh
   ```

## Результаты

Результаты будут сохранены в папке `results/` в формате:
- `hysteria2_results_YYYYMMDD_HHMMSS.txt`

Каждый файл содержит:
- Статистику по странам
- Статистику по скорости
- Детальный список всех рабочих конфигов с ping, скоростью и страной

## Что делает скрипт?

1. ✅ Загружает конфиги из 38+ источников
2. ✅ Фильтрует только Hysteria2 конфиги (hysteria2:// и hy2://)
3. ✅ Проверяет работоспособность каждого конфига
4. ✅ Измеряет ping и скорость соединения
5. ✅ Определяет страну сервера
6. ✅ Сохраняет результаты в текстовый файл

## Время выполнения

Время выполнения зависит от количества найденных конфигов:
- ~1-5 минут для 100 конфигов
- ~5-15 минут для 500 конфигов
- ~15-30 минут для 1000+ конфигов
